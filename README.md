# Node.js + Express Starter Project

This is a **starter template** for Node.js projects using:

- **Express** – Web framework
- **Mongoose** – MongoDB ODM for MongoDB
- **dotenv** – Environment variable management
- **Morgan** – HTTP request logging
- **Nodemon** – Automatic server restart for development

It includes a clean folder structure and ready-to-use configuration.

---

## Project Setup

### 1. Initialize npm

```bash
npm init -y
```

### 2. Install dependencies

```bash
npm install express mongoose dotenv morgan cors jsonwebtoken bcrypt
```

### 3. Install development dependencies

```bash
npm install nodemon --save-dev
```
